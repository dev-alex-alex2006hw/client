scenes

Provides methods getting scene metadata.

Methods

(inner) get(scene, options) → {Promise.<Object>}

Get metadata for a single scene.

Parameters:
Name Type Description
scene Object | string

An object with scene id and type properties. If a string is provided, it is assumed to be the id, and the type will be set to 'ortho'.
options Object

Options.

Properties
Name Type Description
augmentLinks boolean

Add API key to links for image resources in the response. True by default.
terminator function

A function that is called with a function that can be called back to terminate the request.
Returns:

A promise that resolves to scene metadata or is rejected with any error.

Type
Promise.<Object>
